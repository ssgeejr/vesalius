# Brown Bag: AI-Assisted Development

A step-by-step series for IT professionals learning to use AI agents to build and edit software.

No coding experience required. You provide the direction. The AI does the typing.

---

## The Goal

By the end of this series, you will have used an AI agent to build a working Python tool that connects to Freshdesk and creates support tickets — entirely through conversation.

---

## Before You Start

Complete the [Pre-Development Setup](docs/PRE_DEV.md) first. You need your environment ready and your `~\.freshdesk\api.key` config file in place before the first step.
